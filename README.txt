thatmarketingguy — multi-page website
======================================

PAGES
  index.html ........ Home
  results.html ...... Portfolio / case studies (filterable)
  resources.html .... 7 free PDFs
  learn.html ........ Paid blueprints + masterclass
  blog.html ......... Blog (placeholder posts)
  about.html ........ About + videos + feed gallery
  contact.html ...... Intent-split contact form
  css/styles.css .... Shared styles (edit once, applies everywhere)
  js/main.js ........ Shared scripts

TO VIEW LOCALLY
  Unzip, then open index.html in a browser. The menu links the pages.

BEFORE PUBLISHING — fill these in
  1. Images: add files to /images (aditya-hero.jpg, aditya-about.jpg,
     feed-1..6.jpg, blog-1..6.jpg). Names must match exactly.
  2. Videos: on about.html, replace VIDEO_ID_1/2/3 with your YouTube IDs.
  3. Case studies (results.html) + testimonials (index.html): replace ALL
     placeholder numbers with your real, verified Meta Ads figures. Get
     client permission before naming brands.
  4. Proof bar numbers (index.html): replace the [X] figures.
  5. Contact form (contact.html): wire to ConvertKit / Formspree / a
     wa.me WhatsApp link.
  6. Social links (footer): add your real Instagram / YouTube / LinkedIn URLs.
  7. Domain: footer says thatmarketingguy.in — confirm this is the domain
     you registered.

DEPLOY
  - Static host (Hostinger / Netlify / Vercel): upload the whole folder.
  - WordPress: best for the blog; rebuild pages with Astra, keep this as reference.
  - Framer: import using the build prompt (separate file).
